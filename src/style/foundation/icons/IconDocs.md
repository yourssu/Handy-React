# Icon

Handy Iconography는 Handy에서 사용하는 아이콘입니다.

- 아이콘 사이즈는 다음 7가지로 제한됩니다.
  - 12px
  - 16px
  - 20px
  - 24px (default)
  - 28px
  - 32px
  - 36px

모든 아이콘 종류는 아래에서 확인할 수 있습니다.

<br />

## 아이콘 사용법

```typescript
import { IcAddFilled } from '@yourssu/design-system-react';

function App() {
  return (
    <IcAddFilled />
  )
}
```
