# Spacing

Handy를 적용하는 모든 프로덕트의 간격을 설정할 때는,  
Handy **Semantic** Spacing에 정의된 값을 사용해야 합니다.

간격 이외에도, UI 상 간격과 연관되는 속성들에 대해서도 동일하게 적용됩니다.  
예를 들어 `border-radius` 를 적용할 때, `theme.semantic.spacing` 에 정의된 값을 사용해야 합니다.

<br />
<br />

## 예시

사용 예시는 아래와 같습니다.

```tsx
import { styled } from 'styled-components';

const StyledDiv = styled.div`
  width: 100px;
  height: 100px;

  border-radius: ${({ theme }) => theme.semantic.spacing.spacingRadiusL}px;
  background-color: ${({ theme }) => theme.semantic.color.bgBrandPrimary};
`;
```
